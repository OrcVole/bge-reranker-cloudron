BGE Reranker

A self-hosted reranking API — there is no web app to use here.

What this is

It scores how well each candidate passage answers a query, using the cross-encoder BAAI/bge-reranker-v2-m3. It is the second stage of retrieve-then-rerank: a retriever returns a shortlist, this reorders it by true relevance, and you keep the best few.

How to use it

Send a query and a list of texts to POST /rerank with your API key as a Bearer token. You get a relevance score for each text, sorted best first.

curl https://YOUR-APP/rerank \
  -H "Authorization: Bearer YOUR_KEY" \
  -H 'content-type: application/json' \
  -d '{"query":"what is panda?","texts":["hi","The giant panda is a bear endemic to China."]}'

Get your key from the app's Terminal (the >_ button in the Cloudron dashboard): cat /app/data/.secrets/keys.env

Endpoints

  • POST /rerank — the reranking API (Bearer key).
  • /docs — interactive API documentation (Cloudron login required).
  • /health — liveness check (open).

Self-hosted, offline, CPU. Unofficial community package, not affiliated with BAAI or Hugging Face.
